restapiJPA
============

Base project for a Spring MVC RESTful API with JPA / MySQL / C3PO persistence

<h3>Features a VERY compact POM.xml file</h3>

<ol>
  <li>Spring 3.2.2</li>
  <li>Spring MVC</li>
  <li>Jackson parser for JSON consumption and response</li>
  <li>Spring-data</li>
  <li>JPA 2.0</li>
  <li>Hibernate 4.2.2</li>
  <li>MySQL</li>
  <li>Joda Time</li>
  <li>Log4j</li>
  <li>slf4j</li>
</ol>
